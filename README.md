# cOwOnversor

A Rust project to convert scientific values ​​efficiently, effectively, and safely.

Project Features:

* **Fast**: Being written entirely in Rust, the cOwOnversor has the advantage of being fast and efficient in its operations.

* **Educational**: The cOwOnversor aims to facilitate the understanding of unit conversion in educational contexts.

* **Ease to Use**: The cOwOnversor is designed to be simple and easy to use, while also being fast, educational, and useful.


___

## Overview
cOwOnversor is a measurement unit conversion platform using a Command Line Interface (CLI), developed by Luana Carvalho and Shizuko Nakano.
